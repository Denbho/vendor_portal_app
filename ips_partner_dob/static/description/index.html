Partner Birthday Wish

This module automatically sends an email to Partners (Customers/Vendors/Employees) on their Birthdays.

Features

Configure Date of Birth for Partner

You can set date of birth for Partner.

Automatically Send Mail for Birthday Wish

Mail will be automatically send to the Partner using template set.

How It Works

Date of Birth Field : You have to set Birthdate of Partner from the partner view.

Send Birthday Email Field : You have to check this field if you wish the birthday wished be sent automatically.

Template

From here a new Template can be created for Partner or use general available template.

Filter & Group

From Partner views (Customer/Vendors) you can group or filter them using birthday filers (Day, Month or Year)
